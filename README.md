# DEXCOM-TEST
### Stack used
1. Node v18.16.0
2. Express v4.18.2
3. React v.18.2.0
4. Next v13.4.10

### Instructions
1. `cd api`
2. `npm install`
3. `npm run start`
4. Open new terminal from root project folder
5. `cd app`
6. `npm install`
7. `npm run env`
8. `npm run start`

### For testing
9. Go to `localhost:3000`
10. `npm run test`
